## Project Description:

Dataset: movies_train.csv, movies_text.csv (from kaggle)

Libraries:
- Numpy
- Pandas
- sklearn
- src

1. Dimentionality Reduction:
    - Word2Vec

2. Classification

    - SVM
    - Neural Network

3. Clustering
    - K-means
    - Gaussian Mixture Models
    - Hierarchical Clustering
 
4. Evaluation:
    - F_score, Precision, Recall
    - Purity
    - ARI
    - Illustration
